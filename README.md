# Musema - Music Session Manager

Musema is a desktop application that helps musicians or students track their practice sessions, analyze time spent on each piece, and visualize progress over time.

This project was created as a personal tool to improve consistency and gain insight into musical practice habits.

## Features

- Local user creation and login

- Add and manage practice pieces

- Track practice sessions in real time

- View daily session history

- View ranked statistics by total practice time

- Monthly activity summary

- Graphical visualization of practice time
